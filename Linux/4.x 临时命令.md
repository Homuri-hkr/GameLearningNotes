### 4.x	临时命令

dd 复制

| 选项        | 作用                                                |
| ----------- | --------------------------------------------------- |
| if=输入文件 | 指定源文件或源设备                                  |
| of=输出文件 | 指定目标文件或目标设备                              |
| bs=字节数   | 指定一次输入/输出多少字节，把这些字节看做一个数据块 |
| count=个数  | 指定输入/输出多少个数据块                           |
